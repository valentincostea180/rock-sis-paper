# ✊ Rock Paper Scissors (JavaScript)

A simple browser-based Rock–Paper–Scissors game built with vanilla JavaScript and basic DOM manipulation.

## 📌 Overview

The game allows a user to play against the computer by clicking buttons.
The computer generates a random choice, and the score updates in real time.

### ⚙️ Features
Button-based input (Rock, Paper, Scissors)
Random computer opponent
Live score tracking
Round results displayed dynamically
First to 5 points wins, then score resets

### ▶️ How to Run
Open the .html file in your browser
Click a button to play
Results and score will appear on the page

### 🧠 Concepts Used
DOM manipulation
Event listeners
Basic game logic
Random number generation
